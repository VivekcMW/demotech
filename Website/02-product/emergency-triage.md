# Emergency & triage

## Hero
**H1:** Casualty moves fast. So does this.
**Subhead:** Triage-first registration, MLC handling, and order sets built for the golden hour.

## Key capabilities
- **Triage-first flow:** Color-coded triage (ESI/CTAS-style) before demographics; unknown-patient registration with later merge.
- **MLC & statutory:** Medico-legal case flagging, injury documentation with body-map annotation, police intimation log, brought-dead workflows.
- **Resuscitation mode:** One-screen critical view — vitals, orders, drugs with weight-based dosing, timer-stamped event log for code documentation.
- **ED tracking board:** Bay-wise patient board with time-in-department, pending results, disposition status (admit/discharge/refer).
- **Referral & transfer:** Inter-facility referral notes, ambulance handoff documentation.
