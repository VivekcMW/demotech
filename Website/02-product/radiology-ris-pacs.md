# Radiology (RIS + PACS)

## Hero
**H1:** Worklist to signed report, with images where the radiologist is.
**Subhead:** RIS + PACS with structured reporting templates and the compliance trail Indian radiology actually needs.

## Key capabilities
- **Modality worklist:** Orders push to CT/MRI/X-ray/USG via DICOM MWL; no re-typing patient demographics at the console.
- **PACS viewer:** Diagnostic-grade web viewer, comparison studies side by side, neutral dark chrome in both UI themes (images are never recolored).
- **Structured reporting:** Template library per study type, voice dictation, normal-study one-click reports, critical-finding flag with referring-doctor notification.
- **PC-PNDT support:** Form-F capture for obstetric ultrasound, statutory register exports — built into the USG workflow, not a side spreadsheet.
- **TAT dashboards:** Pending-report aging, radiologist productivity, modality utilization.
